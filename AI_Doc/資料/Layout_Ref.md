# 資料 (Materials/Lore) レイアウト構成案

このドキュメントは、世界観設定やアイテム図鑑などの資料画面の基本レイアウト定義です。

## 1. ヘッダー構成 (Header)

### デザイン要素
- **左側**: ノートアイコン (`NotebookPen`) - 内部パラメータやJSONデータの表示トリガー
- **中央**: 資料タイトル（例：「世界地図」「魔物図鑑」）
- **右側**: 閉じるボタン (`X`) または 「戻る」ボタン (`ArrowLeft`)

### UI Layout
| 位置 | 要素 | アイコン (Lucide) |
| :--- | :--- | :--- |
| Left | Data/Info | `NotebookPen` |
| Center | Title | Text |
| Right | Back/Close | `ArrowLeft` or `X` |

## 2. コンテンツエリア (Content)

- **メインビュー**: 画像、リッチテキスト、表形式のデータ表示。
- **JSONビュー**: ヘッダーのノートアイコンからアクセス。資料の元データのJSONを表示。

### Data Structure Example
```json
{
  "material_id": "world_lore_01",
  "title": "古の王国",
  "content": "数千年前、この地を統治していた王国...",
  "tags": ["history", "lore"]
}
```

## 3. デザイン指針
- カードレイアウトを採用し、スクロール可能なグリッドまたはリスト形式で表示。
- 文字サイズは読みやすさを重視（16px以上）。
- カラーパレットは古文書をイメージしたセピア、または近未来的なブルー/ブラック。
