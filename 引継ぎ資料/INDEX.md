## 🎯 用途別ガイド

### 新規参画者向け

1. **[開発環境セットアップガイド.md](./開発環境セットアップガイド.md)** を読んで環境構築
2. **[データ構造＆API仕様書.md](./データ構造＆API仕様書.md)** でアプリの仕組みを理解
3. 実際にローカルでアプリを動かして動作確認

### デザイナー向け

1. **[React移行_レイアウト仕様書.md](./React移行_レイアウト仕様書.md)** で全画面のデザイン仕様を確認
2. デザイントークン（カラー、フォント等）の確認
3. 必要に応じてFigma等でモックアップ作成

### React移行担当者向け

1. **[React移行_レイアウト仕様書.md](./React移行_レイアウト仕様書.md)** でコンポーネント構成を把握
2. **[データ構造＆API仕様書.md](./データ構造＆API仕様書.md)** で状態管理・API仕様を確認
3. **[開発環境セットアップガイド.md](./開発環境セットアップガイド.md)** のReact移行セクションを参照

### 既存コード保守担当者向け

1. **[データ構造＆API仕様書.md](./データ構造＆API仕様書.md)** でデータフォーマットを確認
2. 既存の `index.html`, `style.css`, `AssetImportScreen.js` を参照しながら修正
3. トラブル時は **[開発環境セットアップガイド.md](./開発環境セットアップガイド.md)** のトラブルシューティングを参照

## ⚡ クイックスタート

最速で始めるには：

```bash
# 1. リポジトリクローン
git clone https://github.com/SHUNSUKE-Ks/NovelGameList_Vol1
cd NovelGame01_vol1

# 2. Dropbox設定（任意）
# src/config.js を編集

# 3. ローカルサーバー起動
# VS Code Live Server を使用、または:
python -m http.server 8000

# 4. ブラウザで開く
# http://localhost:8000
```
